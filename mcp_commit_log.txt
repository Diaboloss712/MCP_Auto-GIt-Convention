## Installation

### 1. Dependency installation

```bash
pip install -r requirements.txt
```

### 2. Environment variable setup

Create a `.env` file and paste the following content:

```env
GITHUB_USERNAME=your_github_username
GITHUB_TOKEN=your_github_token
TARGET_PATH=C:/path/to/your/repository
OLLAMA_URL=http://localhost:11434
LOG_FILE=mcp_commit_log.txt
```

**Required environment variables:**
- `GITHUB_USERNAME`: GitHub username
- `GITHUB_TOKEN`: GitHub Personal Access Token (repo access required)

**Optional environment variables:**
- `TARGET_PATH`: Path to your Git repository (default: current project directory)
- `OLLAMA_URL`: Ollama server URL (default: http://localhost:11434)
- `LOG_FILE`: Log file path (default: mcp_commit_log.txt)

### 3. Install and run Ollama

```bash
# Olla
```
